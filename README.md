# leetcode 答え置き場

## install

elixirでやりたいとき
```
npm install -g https://github.com/ThatHurleyGuy/leetcode-cli#add-elixir
```

ログイン
```
leetcode user -c
# (ユーザ名を入力)
# (leetcodeにログイン済みのブラウザセッションからcookieをコピペしてきて入力)
```

問題開く
https://leetcode.com/problemset/all/

```
leetcode show 2 -g -l elixir
```

iexひらく
```
iex ----.ex
```

テスト
```
leetcode test ----.ex
```

提出
```
leetcode submit ----.ex
```
